For building and running the application you need :)

* Node.js 
* Npm 

<h3>Installation</h3>

```
$ git clone git@github.com:Emma-Hyejin/todo-v2.git
$ npm i
$ npm run start
```

## 아키텍쳐
프론트와 백엔드가 어떻게 소통하는지 / 디렉토리 구조는 tree를 이용해 그려줄 수 있다. 
```bash
├── README.md
├── package-lock.json
├── package.json
├── client : 프론트엔드
│   ├── README.md
│   ├── components : 기능 구현 폴더
│   │   ├── DarkMode
│   │   ├── Item
│   ├──  pages : 각 화면 단 구현 폴더
│   │   ├── App.tsx
│   │   ├── index.tsx
│   │   ├── Home.tsx
│   │   └── Main.tsx
│   └──  styles : global 전역 CSS 구현 폴더
│       └── style.d 
│       └── theme 
└── server : 백엔드
```
